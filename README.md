# HR Workforce Analysis

## Overview
This project analyzes the company’s workforce structure, focusing on employee distribution, profile, and compensation.

## Key Findings
The company shows a balanced structure across departments in terms of headcount, average age (~42 years), and salary.

However:
- Female representation is low (~33%) across all departments
- There is concentration of employees in non-core areas
- Some high-paying roles lack full benefits coverage
- Salary ranges show high variation within the same roles

## Conclusion
Overall, the structure is stable, but there are opportunities in workforce allocation, diversity, and compensation consistency.
